# Data Cleaning and Visualization

A collection of projects and exercises for learning data cleaning, analysis, and visualization techniques using Python libraries like Pandas, Matplotlib, Seaborn, and other data science tools.

## Prerequisites

- Python 3.x or higher
- Git
- Python libraries: Pandas, Matplotlib, Seaborn, NumPy (and others as needed)

To verify your Python installation:

```bash
python --version
```
or
```bash
python3 --version
```

## Installation

Clone the repository:

```bash
git clone https://github.com/mosesamwoma/data-cleaning-and-visualization.git
cd data-cleaning-and-visualization
```

Install required Python libraries:

```bash
pip install -r requirements.txt
```

## Notes

- Code includes comments and explanations for learning purposes
- Projects cover data cleaning, exploratory data analysis (EDA), and visualization techniques
- Datasets used for practice are included or referenced in the project files
